## How to Link in Fibery

1. Open a to-do view filtered by Public ID
2. Select the child to-dos for one parent (e.g., 33139-33145)
3. Use bulk edit to set "Blocked by" to the parent to-do
4. Repeat for each parent group

**Tip:** After linking, rename the fields:
- "Blocked by" → "Parent"
- "Blocking" → "Sub-tasks"
